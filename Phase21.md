## 已確認的設計決策（開發前經審核）

| 項目 | 決策 |
|---|---|
| 社團 vs 舊幫會 | **擴展現有 Phase 13 幫會系統**（沿用 guilds collection，舊職級自動兼容：龍頭→話事人、成員→四九仔） |
| 武器強化上限 | **+15**；失敗只損失金錢與強化石（**不降級、不損毀**）；連續失敗 5 次保底必成功 |
| 組隊 | **5 人**上限；擊殺經驗**範圍內隊員各得全額**（不均分） |
| 換旗保護期 | **30 分鐘** |
| 守衛 | Lv1~10、招聘成本遞增（$1,000~$50,000，社團資金支付）；社團 Lv N 可招 Lv N×2 守衛 |
| 守衛雙模式 | **持有社團成員攻擊 = 練功模式**（守衛 30 秒重生、經驗加成、強化石高掉落）；**非成員攻擊 = 佔領模式**（守衛不重生，全滅即換旗） |
| 任免 | 話事人任免全部；坐館任免紅棍以下；名額：坐館1、紅棍2、白紙扇1、草鞋2 |
| 貢獻度比例 | ⚠️ 暫定 **100 金 = 1 貢獻、無上限**（該題回答被守衛規則覆蓋，數值在 `newSystems.ts` 的 `SOCIETY_CONFIG` 可隨時調整，如需修改請告知） |

所有數值集中在 `packages/shared/src/newSystems.ts` 的 `*_CONFIG` 常數。

## 遊戲內快捷鍵（Client）

| 鍵 | 功能 |
|---|---|
| **U** | 背包 / 武器強化面板（裝備、強化、入倉） |
| **G** | 社團面板（職級/任免/捐獻/倉庫/社團商店/解散） |
| **P** | 組隊面板（邀請/接受/離開） |
| **T** | 地盤面板（列表/招聘守衛；換旗全自動，打死全部守衛即觸發） |
| **K** | 發放測試套件：唐刀 +0 ×1、強化石 ×20、$100,000 |
| **N** | 在玩家附近生成 5 隻街頭混混（練功測試用） |

> 測試道具不需在 Dashboard 手動建立：K / N 鍵即可發放測試武器、強化石、金幣與測試怪。

## 各系統測試步驟

### 1. 武器升級系統
1. 登入遊戲，按 **K** 領取測試套件。
2. 按 **U** 開背包：點「裝備」唐刀 → 上方顯示攻擊加成。
3. 點「強化 +1」連續強化：+1~+5 必成功；+6 起有機率失敗。
4. 驗證失敗懲罰：失敗時只扣金錢與強化石，武器等級不變（面板顯示連敗次數）。
5. 驗證保底：連續失敗 5 次後下一次必成功（訊息顯示「保底觸發！」）。
6. 驗證上限：強化到 +15 後按鈕顯示「已滿級」。
7. 按 **N** 生成測試怪並攻擊，確認傷害 = 30 + 武器攻擊加成（看 server log）。
8. **Dashboard → 系統資料 → 武器強化紀錄**：每次強化都有一筆紀錄（成功/失敗/保底/消耗）。

### 2. 社團系統
1. 用現有幫會介面建立幫會（或已有幫會）。
2. 按 **G** 開社團面板：確認舊「龍頭」顯示為「話事人」。
3. 捐獻：輸入 1000 → 獲得 10 貢獻度；確認社團資金 +$1000、社團經驗 +10。
4. 任免：話事人在成員列表用下拉選單改職級；驗證名額限制（坐館只能 1 人）與權限（四九仔改別人職級會被拒）。
5. 倉庫：背包按「入倉」存入物品；只有話事人/坐館/白紙扇能「取出」（用四九仔帳號測試會被拒——權限矩陣）。
6. 社團商店：用貢獻度兌換強化石/西瓜刀；貢獻度不足會被拒。
7. 社團升級：累計捐獻 500 貢獻 → 社團 Lv2（全服廣播），成員上限提高、可招聘守衛等級提高。
8. **Dashboard → 系統資料 → 社團**：檢視職級、貢獻度、倉庫、資金。

### 3. 佔領地盤系統
前置：先在 **Dashboard → 地圖管理 → 地盤設置** 畫一塊地盤（見下節）。地盤建立後**伺服器會自動補上 5 個 Lv1 中立守衛**，不需任何手動操作即可測試。
1. 走進地盤範圍，按 **T**：該地盤顯示「你在此地盤內」，並可看到 5 隻 Lv1 中立守衛。
2. **換旗全自動**：擊敗地盤內全部守衛 → 若擊殺最後一名守衛時你有社團，該地盤**自動**換旗成為你們社團的地盤（全服廣播、30 分鐘保護期）；若你沒有社團，地盤保持中立，5 分鐘後自動補滿守衛（不需任何人手動處理）。
3. 招聘守衛（僅限已換旗、自己社團持有的地盤）：按 **T** 對自家地盤選守衛等級按「招聘守衛」→ 從社團資金扣款、守衛出現在地盤中央（可先捐獻充實資金；社團 Lv1 最高招 Lv2 守衛）。換旗當下守衛位是空的，需重新招聘。
4. **練功模式**：持有社團成員攻擊自家守衛 → 擊殺後獲得加成經驗（80×守衛等級）、金幣（20% 歸社團資金）、35% 強化石掉落；守衛 30 秒後重生。
5. **佔領模式**：非持有社團（或無社團）玩家攻擊守衛（含中立守衛）→ 保護期內會被擋（中立地盤無保護期，隨時可打）；守衛死後**不重生**；全部守衛被清空後依上述規則自動換旗或保持中立。
6. 守衛 AI：非持有社團玩家進入地盤會被守衛追擊，守衛不會離開地盤範圍。
7. 重啟 server → 地盤歸屬與守衛配置保留（Firebase 持久化）。
8. **Dashboard → 系統資料 → 地盤**：檢視歸屬、保護期、每個守衛位的狀態。

### 4. 練功掉落
1. 按 **N** 生成測試怪，擊殺數隻。
2. 掉落中應出現「強化石」（約 15%；守衛為 35%）。
3. 拾取後按 **U** 確認強化石進入背包並可用於強化。

### 5. 組隊系統
需兩個帳號（兩個瀏覽器）。
1. A 按 **P** → 輸入 B 的角色名稱 → 邀請。
2. B 畫面自動彈出邀請 → 接受 → 兩人面板顯示隊伍成員（👑 = 隊長）。
3. A 按 **N** 生成怪並擊殺 → B 在 40m 範圍內會收到「[隊伍] 獲得 50 經驗」且是**全額**（雙方各 +50）。
4. B 走遠（>40m）再擊殺 → B 不獲得經驗（範圍限制）。
5. 離開隊伍 / 斷線 → 隊伍自動更新；剩 1 人自動解散。
6. **Dashboard → 系統資料 → 組隊**：檢視進行中隊伍快照。

### 6. Dashboard 地盤設置模組
路徑：Dashboard 側邊欄 → **地圖管理 → 地盤設置**（`/map/territory`，與地圖編輯器同屬 Map Management，需 `map.view` 權限；編輯操作需 `map.edit`）。

**地圖顯示**：與地圖編輯器共用同一份地圖來源 — `predev`/`prebuild` 的 copy-maps 腳本會自動把 `packages/client/public/maps` 複製到 dashboard 的 `public/maps`，無需手動處理（也可用 `NEXT_PUBLIC_MAP_BASE_URL` 指定外部來源；載入失敗時退回平面網格，仍可繪製）。

1. **新增領地**：點「➕ 新增領地」→ 在地圖上逐點 click 連線 → 點回綠色起點自動封閉 → 輸入名稱完成。
2. **落點防呆**：嘗試把點落在既有點上、或讓新邊跨過既有邊 → click 被擋掉並提示。
3. **編輯**：列表選取領地 → 「✏️ 編輯頂點」→ 拖曳橘色頂點（**不可增刪頂點**）；拖到會造成頂點重疊/邊交叉/與其他領地重疊的位置不會生效（放開即回彈）。
4. **回溯**：編輯中按「↩️ 回溯」→ 恢復到進入編輯時的快照；「💾 儲存修改」寫入 Firebase。
5. **重疊檢查**：畫第二塊領地與第一塊相交 → 封閉時被拒。
6. **列表**：右側列表選取（點位密集時用）、更名、刪除。
7. **顯示開關**：「🙈 隱藏大廈」隱藏所有 B 開頭的建築 mesh，方便看清街道。
8. **鏡頭**：Space+左鍵拖曳平移、右鍵拖曳旋轉、滾輪縮放。
9. 確認新領地**沒有擁有者欄位**（一律中立），守衛上限固定 10。
