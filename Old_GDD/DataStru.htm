遊戲中使用了 SQL Database 作為儲存進度資料的工具。以下是各個資料名稱及所需空間：

＊所有欄位是以 Byte 為單位
資料名稱 所需空間 解說
UniqueID 16 Bytes 獨立編號，用來檢查有多少下線
UpLine 16 Bytes 老大的獨立編號
District 20 Bytes 玩家居住地區，可自由輸入地區名稱
Name 16 Bytes 角色姓名
Level 2 Bytes 等級
HP 4 Bytes 生命值，當到達零時便會死亡
MaxHP 4 Bytes 生命值上限
Exp 4 Bytes 經驗值
MaxExp 4 Bytes 經驗值上限，當滿檔時便會上升一級
Money 4 Bytes 金錢
Killed 2 Bytes 殺人數目
Died 2 Bytes 死亡次數
Weapon 2 Bytes 裝備了的武器
Inventory 32 Bytes 物品欄
